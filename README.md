# Spam Message Detector

This project is a **regular expression-based spam filter** that detects spammy messages in real time. Users can type a message into the text area and click **"Check message"** to see if it might be spam.

## Features

- Detects messages containing:
  - Requests for help (`please help`, `assist me`)
  - Dollar amounts (`100 dollars`, `5 million dollars`)
  - Free money scams (`free money`, `fr33 mon3y`)
  - Stock alerts (`stock alert`, `st0ck al3rt`)
  - Spammy greetings (`dear friend`, `d3ar fr13nd`)
- Supports variations using letters replaced by numbers.
- Built with **HTML, CSS, and JavaScript**.
- Provides a simple and interactive user interface.

## How to Use

1. Open `index.html` in your browser.
2. Type your message in the text area.
3. Click **Check message** to see if it's spam.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Regular Expressions (Regex)
